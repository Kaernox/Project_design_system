akk-multiselect

The akk-multiselect directive takes a list of object (in the items property) and the field to be shown from those objects in the modal that will appear, selected objects will be removed from the items array and added to the selected array.

Scope Required Format Example
label X string this is a multiselect
model X string model
placeholder X string My placholder
items ✓ array []
selected ✓ array []
field ✓ string name
event X string functionEvent
