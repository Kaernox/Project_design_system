akk-multiselect

Description

The akk-multiselect directive takes a list of object (in the items property) and the field to be shown from those objects in the modal that will appear, selected objects will be removed from the items array and added to the selected array.

Data

{{ view.multiselect.selected| json }}

Properties

Scope Attribute Required Format Example
label clear String "this is a multiselect"
model clear String "model"
placeholder clear String "My placholder"
items check Array [{id: 1, name: 'test 1', value: true}]
selected check Array [{id: 2, name: 'test 2', value: false}]
field check String "name"
event clear String "functionEvent"
