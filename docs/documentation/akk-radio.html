akk-radio

Description

The akk-radio is a set of radio buttons where options are specified in the options property as objects and the selected option will be in the model.
In property, you specify a field to be selected, model will contain the field vaue of the selected object, if no property has been specified, the model will contain the entire selected object, this will give you freedom to use primitives types as well as objects.
The akk-radio directive requires a all objects in options to have a property name so it can use it to display radio buttons.
