akk-checkboxList

Here we have our akk-checkbox-list directive, it's usefull when you have a list of options that you want to present to your users as a checkbox list, it taks the options to show the user and the model that will contain the selected items.

Scope Required Format Example
label X String "Checkbox List"
elementclass X String "class"
req X Boolean true
model ✓ Array ["test1", "test3"]
options ✓ Array ["test1", "test2", "test3", "test4"]
value X Boolean true
display X Boolean false
event X String "alert"
