akk-checkbox

Here we have our akk-checkbox directive, it takes three values, the label, the model object and the property to take from that model object. The property will contain true if akk-checkbox is checked, or false.

Scope Required Format Example
label ✓ String "Do accept tde conditions of use?"
alignment X String center
