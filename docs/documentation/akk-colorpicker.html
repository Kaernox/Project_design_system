akk-colorpicker

The akk-colorpicker directive is an improvement of the angular colorpicker directive that provide a visual view of colors, it takes the model which will contain the color chosen by the user in a 6 digit hex format.

Scope Required Format Example
label ✓ string test color picker
req X boolean true
model ✓ string #CC0000
options X array []
event X string function
