akk-tree

Description

The akk-tree directive takes a list of items with children (at any depth) and diplays them in a tree like format, the selected items of the tree will be stored within the model property.

Data

{{ view.tree.model| json }}

Properties

Scope Required Format Example
title check String "My tree"
items check Array [{label: 'example', value:'example_value}]
model check Array [{label: 'example', value:'example_value}]
options check Object {debug: false, label: 'label',icon: 'label'}
multiple check Boolean true
selectable check Boolean false
eventClick clear String "functionCLick"
eventUpdate clear String "functionUpdate"
