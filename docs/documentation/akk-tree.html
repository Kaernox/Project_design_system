akk-tree

The akk-tree directive takes a list of items with children (at any depth) and diplays them in a tree like format, the selected items of the tree will be stored within the model property.

Scope Required Format Example
title ✓ string My tree
items ✓ array []
model ✓ array []
options ✓ objet {debug: false, label: 'label',icon: 'label'}
multiple ✓ boolean true
selectable ✓ boolean false
eventClick ✓X string treeSelectItem
eventUpdate X string functionUpdate
