akk-textarea

Description

The akk-textarea directive allows your users to type some text and stores it in the model property, you can increase the length of the text area as you like by dragging the bottom-right corner up and down.

Data

{{ view.textareaValue ? view.textareaValue : 'Start typing to see the result' }}

Properties

Scope Required Format Example
label check String "Input your text here"
