The akk-datagrid takes an array of objects and an array of columns names and displays it, the datagrid is sortable using any column, just click on the column name to sort, clicking multiple times will toggle the sort type between ascending and descending.
You can add a checkbox for the datagrid and in all the entry, this serve to pass parameter and select multiple choice in the datagrid.
You can also add an expense button like in this exmaple at the right of the list and add multiple function to it. In this example we you can see a button edit and delete who open a modal and says something.
