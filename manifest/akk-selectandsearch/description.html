Description

The akk-selectandsearch directive accepts an array of objects as input as well as how you want to display every object ('fields' attribute) in the modal that pops off, the object passed to the model attribute will contain the id of the selected object, the full list of objects is displayed using the akk-datagrid.
To start Selecting items, click on the 3 dots on the right.

Data

{{ view.selectAndSearch.selectedId | json }}
