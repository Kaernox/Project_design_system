The akk-select directive provides a modal with options to select from. Options are specified in the options property and the selected item will be stored in the model property.

Scope Required Format Example
label X String "test select"
elementclass X String "class"
req X Boolean true
model ✓ String "value 1"
options ✓ Array [{id: 1, key: 'one', label: 'one'}]
value ✓ String "id"
display ✓ String "name"
update X String "functionUpdate"
defaultDisplayEnabled X Boolean false
defaultDisplay X Boolean false
